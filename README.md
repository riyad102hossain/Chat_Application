# Chat Application

## About the Project
This Chat Application allows users to communicate with each other in real time with features like message history, user authentication, and presence indicators (online/offline status). Built with a scalable architecture, this project leverages modern web technologies to ensure smooth, real-time communication. The project follows a client-server model where the front end is developed using Angular and the backend with ASP.NET Core Web API, ensuring seamless integration with the database using Entity Framework.

## Key Features
- **User Authentication**: Secure login, logout, and registration functionalities.
- **Real-time Messaging**: Users can send and receive messages instantly.
- **Online/Offline Status**: Shows the active status of users in real time.
- **Search Users**: Search for other users to initiate conversations.
- **Persistent Chat History**: Chat history is stored in the database and retrieved upon request.
- **Responsive Design**: User-friendly interface optimized for both desktop and mobile devices.

## Technologies Used
- **Backend**: 
  - ASP.NET Core Web API
  - C#
  - Entity Framework Core (ORM for database access)
  - SignalR for real-time communication
  - Microsoft SQL Server for data storage

- **Frontend**: 
  - Angular (for single-page application development)
  - TypeScript
  - HTML/CSS for UI

- **Database**: 
  - Microsoft SQL Server
